💾 Exportar / Importar datos

Como la app corre localmente (sin servidor), los datos se guardan en el navegador.
Para no perder datos al cambiar de computadora o navegador, exportá un archivo JSON.

📥 Exportar datos

Descarga un archivo JSON con todos los datos del sistema (clientes, créditos, config, usuarios, etc.)

📂 Importar datos

Cargá un archivo JSON exportado previamente. Reemplaza todos los datos actuales.

💡 Flujo recomendado: Al terminar de trabajar → Exportar datos → Guardar el JSON en tu computadora. La próxima vez → Abrir la app → Importar datos.
